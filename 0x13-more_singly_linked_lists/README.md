2. **1-listint_len.c**
- a function that return the number of nodes in a list

3. **2-add_nodeint.c**

4. **3-add_nodeint_end.c**

5. **4-free_listint.c**
- a function that frees a listint_t list

6. **5-free_listint2.c**
